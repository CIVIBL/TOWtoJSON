# TOW List Converter

Convert Warhammer: The Old World army lists to importable JSON for army builder apps.

**[Use it live](https://civibl.github.io/TOWtoJSON/)**

## What it does

Paste any text army list and get a JSON file you can import directly into Old World Builder (or New Recruit, coming soon). No account needed, runs entirely in your browser.

## Supported input formats

- BCP (Best Coast Pairings) text export
- Old World Builder text export
- New Recruit text export
- Any plain text containing recognizable unit names and points

The parser uses faction data as a dictionary to find units in any text format — it doesn't require a specific structure.

## Supported output formats

- **Old World Builder** — ready to import
- **New Recruit** — coming soon

## Supported factions

All 17 factions: Beastmen, Bretonnia, Chaos Dwarfs, Daemons of Chaos, Dark Elves, Dwarfs, Empire, Grand Cathay, High Elves, Lizardmen, Ogre Kingdoms, Orcs & Goblins, Skaven, Tomb Kings, Vampire Counts, Warriors of Chaos, Wood Elves.

## How to use

1. Paste your army list text
2. Select output format (OWB or NR)
3. Click "Convert & Download"
4. Import the JSON file into your army builder
